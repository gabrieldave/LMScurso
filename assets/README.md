# Assets

Esta carpeta contiene los recursos de la aplicación (iconos, splash screen, etc.).

Para generar los assets correctamente, puedes usar herramientas como:
- https://www.appicon.co
- https://www.favicon-generator.org
- O crear manualmente con herramientas de diseño

Los assets requeridos son:
- icon.png (1024x1024)
- splash.png (1242x2436)
- adaptive-icon.png (1024x1024)
- favicon.png (48x48)
